# design_bondi
Design Bondi هو مشروع تصميم باستخدام Bootstrap 5، تم تقديمه بواسطة أكاديمية الزيرو. يهدف هذا المشروع إلى تعليم كيفية استخدام Bootstrap لإنشاء تصميمات ويب متجاوبة وجذابة. يتضمن المشروع عدة دروس تغطي مختلف جوانب التصميم، مثل إعداد المشروع، إنشاء أقسام الهبوط، وإضافة ميزات مختلفة12.  يمكنك العثور على المزيد من المعلومات ومتابعة الدرو
